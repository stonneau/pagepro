A matter of some gravity

Description

Il s'agit d'un prototype de jeu développé pour un entretien d'embauche.

Il se compose :

  • D'un fichier texte README contenant les instructions nécessaires pour jouer;
  • d'un fichier texte contenant les réponses à un questionnaire sur le gameplay des différents jeux prince of persia;
  • du fichier GameDesignTest.pdf. Il contient les spécifications d'un jeu de plateforme dont les contraintes sont également détaillées;
  • d'un prototype du jeu de plateforme spécifié, "a matter of some gravity";
  • d'une vidéo démontrant comment franchir le niveau;
  • d'un screenshot du niveau de démo.
