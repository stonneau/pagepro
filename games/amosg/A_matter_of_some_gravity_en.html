Description

Game prototype made in a few days as a hiring test for Gameloft. Several constraints were imposed on the gameplay.

The attached file consists in:

  • A README file with playing instructions;
  • a text file containing the answers to a questionnaire about the game design of the prince of persia series;
  • A document word GameDesignTest.pdf, containing the specifications of the game, and the imposed constraints;
  • The game prototype "a matter of some gravity";
  • A video showing the solution of the demo level;
  • A screenshot of the demo level
